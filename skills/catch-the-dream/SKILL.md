# Catch the Dream 🌙

## Purpose

You have ~30 seconds after waking before a dream fades. This skill helps you capture dreams quickly via voice or text, then structures them into a searchable log with symbol tracking.

## When to Activate

- Waking up from a vivid dream
- "I had the weirdest dream..."
- "What does this dream mean?"
- Reviewing dream patterns over time
- Noticing recurring symbols or themes

## The 30-Second Window

**The problem:** Dreams evaporate fast. By the time you open a notes app, half the dream is gone.

**The solution:** Capture first, structure later.

```
[Wake up] → [Voice note or quick text dump] → [AI structures it] → [Pattern emerges over time]
```

## Quick Capture Template

**For voice notes or raw text dumps:**

Just speak/type everything you remember. Don't worry about order or making sense.

**AI will extract:**
- Setting/Location
- People present
- Key events
- Emotions felt
- Symbols/Objects
- Any dialogue

## Structured Dream Log Format

```markdown
## Dream: [Auto-generated title]
**Date:** YYYY-MM-DD
**Time:** ~HH:MM (when you woke)
**Vividness:** 1-5 ⭐

### Setting
[Where the dream took place]

### People
- [Person 1] — [relationship/role]
- [Person 2] — [relationship/role]

### Events
1. [What happened first]
2. [Then this]
3. [Then this]

### Emotions
[How you felt during the dream]

### Symbols
| Symbol | Possible Meaning |
|--------|-----------------|
| [Object/Image] | [Common interpretation + personal context] |

### Raw Notes
[Original voice/text dump preserved]

### Tags
#recurring-[symbol] #emotion-[feeling] #person-[name]
```

## Symbol Tracking

| Symbol | Count | First Appeared | Context |
|--------|-------|----------------|---------|
| Water | 12 | 2026-01-15 | Often turbulent when stressed |
| Flying | 8 | 2026-02-03 | Usually positive, freedom |
| Teeth falling | 3 | 2026-03-10 | Before big presentations |

## Common Dream Symbols Reference

| Symbol | Common Meanings |
|--------|-----------------|
| **Water** | Emotions, unconscious, cleansing |
| **Flying** | Freedom, ambition, escape |
| **Falling** | Loss of control, anxiety |
| **Teeth** | Confidence, appearance, communication |
| **Being chased** | Avoidance, fear, running from something |
| **House** | Self, psyche, different rooms = different aspects |
| **Death** | Transformation, endings, change |
| **Nakedness** | Vulnerability, exposure, authenticity |
| **Exams** | Self-evaluation, fear of failure |
| **Lost/Late** | Anxiety about direction, missed opportunities |

**Note:** Personal context matters more than generic meanings. Track YOUR patterns.

## Example

**Input (voice note transcript):**
```
Okay so I was in my old high school but it was also kind of my office? 
And my mom was there but she was my boss. I was late for something, 
running through hallways that kept getting longer. Then I was flying 
over the building and felt relieved. Woke up feeling weird but okay.
```

**Output:**
```markdown
## Dream: The Endless Hallway
**Date:** 2026-03-24
**Time:** ~6:30 AM
**Vividness:** 4 ⭐

### Setting
High school merged with current office — identity/role confusion

### People
- Mom — appeared as boss figure (authority, expectations)

### Events
1. In hybrid school/office space
2. Running late for unknown obligation
3. Hallways kept extending (obstacle, frustration)
4. Transitioned to flying over building
5. Felt relief, then woke

### Emotions
- Anxiety (running late, endless hallways)
- Relief (flying, escape)
- Residual unease on waking

### Symbols
| Symbol | Possible Meaning |
|--------|-----------------|
| Old high school | Past self, old pressures, formative experiences |
| Office merge | Current life blending with past patterns |
| Mom as boss | Authority figure, approval seeking, expectations |
| Endless hallways | Feeling stuck, goal keeps moving |
| Flying | Escape, freedom, rising above problems |

### Tags
#recurring-flying #recurring-school #emotion-anxiety #emotion-relief #person-mom
```

## Weekly Dream Review

Every week, review:

1. **Frequency** — How many dreams logged?
2. **Themes** — What kept appearing?
3. **Emotions** — Mostly positive, negative, mixed?
4. **Connections** — Any link to waking life events?

```markdown
## Week of YYYY-MM-DD Dream Summary

**Dreams logged:** X
**Most common symbol:** [symbol] (appeared X times)
**Dominant emotion:** [emotion]
**Notable pattern:** [observation]

**Waking life correlation:**
- [Dream theme] might relate to [life event]
```

## Anti-Patterns

| ❌ Don't | ✅ Do Instead |
|----------|---------------|
| Wait until fully awake to log | Capture immediately, even half-asleep |
| Try to interpret while capturing | Log first, interpret later |
| Dismiss "boring" dreams | All dreams have value, log everything |
| Force meaning onto symbols | Let patterns emerge naturally |
| Only log vivid dreams | Fragments count too |

## Response Principles

When helping with dreams:

1. **Preserve the raw first** — Never lose the original capture
2. **Structure without over-interpreting** — Present possibilities, not certainties
3. **Track patterns over time** — One dream means little, patterns mean everything
4. **Personal context > generic meanings** — User's associations matter most
5. **No judgment** — Dreams can be weird, violent, sexual, nonsensical. That's normal.
